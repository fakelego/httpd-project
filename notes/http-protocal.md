# HTTP 协议学习笔记

## HTTP 请求格式
```
GET /path HTTP/1.1 Host: localhost User-Agent: curl Accept: /
```

## HTTP 响应格式
```
HTTP/1.0 200 OK Content-Type: text/html Content-Length: 123

<html>...</html> ```

## 状态码
- 200 OK
- 404 Not Found
- 500 Internal Server Error
- 501 Not Implemented

## 方法
- GET: 获取资源
- POST: 提交数据
- PUT: 更新资源
- DELETE: 删除资源
